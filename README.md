##  Visão Geral do Projeto

Este projeto é um projeto do rico universo de **The Witcher**. O foco é na lógica de jogo essencial, utilizando **JavaScript** no ambiente Node.js.

###  Contexto

Este projeto foi desenvolvido por **Vinicius Mendes da Silva** como um **projeto individual** para o curso de **Ciências da Computação** na faculdade **SPTech **.

##  Guia de Configuração e Uso

Siga estes passos para ter o projeto rodando em sua máquina local.

### 1. Clonagem do Repositório

Primeiro, obtenha uma cópia do código-fonte utilizando o `git clone ` e navegue até o diretório do projeto:

```bash
git clone https://github.com/VinnyMendess/The-Witcher-FanSite
cd The-Witcher-FanSite

### 2 Instale as Dependecias
npm install
# ou
npm i

### 4. Ajuste o script SQL dentro do repositório no seu sql, e troque o .env.dev

### 5. Rode o código
npm run start
# Verifique o seu package.json para confirmar se o script de inicialização é 'start'.
